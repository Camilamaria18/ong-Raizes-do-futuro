# ONG Raízes do Futuro

Este projeto é o **site institucional da ONG Raízes do Futuro**, desenvolvido como parte das atividades da disciplina de Web Design. O site tem como objetivo apresentar informações sobre a ONG, seus projetos e oferecer um formulário de cadastro para interessados em participar ou apoiar as ações.


## Estrutura do Projeto

O projeto está organizado da seguinte forma:

- **index.html**: Página inicial com banner, informações sobre a ONG e acesso às demais seções.  
- **projetos.html**: Página dedicada a apresentar os projetos da ONG em formato de **cards responsivos**.  
- **cadastro.html**: Página com **formulário de cadastro** estilizado e validado via JavaScript.  
- **css/style.css**: Arquivo principal de estilos, com sistema de cores, tipografia hierárquica e espaçamentos modulados.  
- **js/script.js**: Arquivo JavaScript responsável pelo menu hambúrguer, validação de formulário e pequenas interações.  
- **images/**: Pasta com todas as imagens usadas no site (banner, projetos etc.).


## Tecnologias Utilizadas

- **HTML5** para a estrutura semântica das páginas.  
- **CSS3** com **Flexbox e Grid** para layout responsivo.  
- **JavaScript** para interatividade (menu mobile e validação de formulário).  
- Sistema de cores e tipografia hierárquica com **paleta de cores consistente** e **espacamentos modulares**.


## Destaques do Projeto

- Menu principal **responsivo**, incluindo **submenu dropdown** e menu hambúrguer para mobile.  
- **Cartões de projetos** com layout flexível e responsivo.  
- **Botões com estados visuais**: hover, focus, active e disabled.  
- Formulários estilizados com validação visual.  
- Sistema de categorização com **tags e emblemas**.  

## Como Visualizar

Para acessar o site, basta abrir o arquivo **index.html** em qualquer navegador moderno. Todos os links internos e imagens estão configurados para funcionar **localmente** ou no GitHub Pages.

## Observações

- O projeto mantém **cores e tipografia destacadas** para títulos e palavras importantes, melhorando a hierarquia visual.  
- Toda a estrutura é responsiva, ajustando-se a diferentes tamanhos de tela, garantindo **acessibilidade e usabilidade**.
